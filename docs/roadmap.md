# Roadmap

## Competition Demo

- One-command demo: tests, compare table, showcase run, report, replay.
- Judge-facing README and competition guide.
- Random, baseline, and rule-aware agents.
- JSONL logs and battle reports.
- Self-contained HTML replay viewer generated by `npm run demo:full`.
- Self-contained HTML comparison arena generated by `npm run demo:full`.
- Tournament-style leaderboard generated by `npm run demo:full`.

## Benchmark Hardening

- Formal JSON schemas for `STATE`, `EVENT`, and JSONL logs.
- Public adversarial seed set with hidden-rule coverage.
- Private and held-out seed workflows for final rankings.
- Stable score breakdown and capability metrics.
- Faster multi-agent evaluation harness.
- CI for tests, full demo execution, generated artifact checks, and showcase artifact upload.

## Reasoning Depth

- Stronger hidden-rule families.
- Seeds that require active experiments.
- Rule-claim explanations in an optional audited mode.
- Constraint-propagation and rule-aware reference agents.

## Viewer And Arena

- HTML replay viewer for single-run judge review.
- Side-by-side agent comparison for bundled demo agents.
- Score delta and event timeline.
- Extended tournament exports across multiple submitted agents.

## Long-Term Position

EchoGrid should grow into an agent-native inference game platform: a small, auditable environment where agents show observation, hypothesis formation, planning, verification, and post-run strategy transfer.
